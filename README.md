# Multiple Layer Perceptron Hands-on using IRIS dataset

## Project Overview
- Implemented a **Multilayer Perceptron (MLP)** from scratch using **NumPy**.
- Dataset: **IRIS** (150 samples, 3 species, 4 features).
- Objective: Classify iris species based on sepal & petal measurements.

---

## Dataset
- **Features:** sepal length, sepal width, petal length, petal width
- **Target Classes:** setosa, versicolor, virginica
- **Number of Instances:** 150
- **Source:** sklearn.datasets.load_iris

---

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## Results
- Loss and Accuracy tracked per epoch
- Loss decreases steadily
- Accuracy increases towards 100% on training set
